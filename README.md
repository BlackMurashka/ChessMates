# ChessMates

Одностраничное приложение для кругового шахматного турнира. Проект подготовлен так, чтобы его можно было сразу публиковать через GitHub Pages, используя встроенный движок Jekyll.

## Быстрый старт

1. Установите Ruby и Bundler (если их ещё нет).
2. Установите зависимости Jekyll:
   ```bash
   bundle install
   ```
3. Запустите локальный сервер GitHub Pages/Jekyll, чтобы протестировать SPA до публикации:
   ```bash
   bundle exec jekyll serve
   ```
4. Откройте `http://127.0.0.1:4000` и проверьте функциональность приложения.
5. Зафиксируйте изменения и отправьте их в репозиторий GitHub. В настройках включите GitHub Pages (deploy from branch) — Jekyll соберёт сайт автоматически.

## Файлы

- `index.html` — само приложение c фронтенд‑логикой.
- `_config.yml` — конфигурация Jekyll/GitHub Pages.
- `Gemfile` — зависимости для локального запуска.

После публикации страница сразу будет готова для ввода участников, заполнения результатов и генерации ссылки для обмена.
